# Tesla-R1-Voice-Control

Voice-controlled Tesla commands for the Rabbit R1.

## Features
- Voice commands for climate control, locking, and more.
- Optimized for the Rabbit R1 display and interface.
- Easy setup via Tesla refresh token.

## Setup
1. Host these files on GitHub Pages.
2. Scan the QR code or paste your Tesla refresh token in the app.
3. Use voice commands like "warm the car" or "lock the car".
